# 📦 ZUSER_ACTIVITY_MONITOR

An **ABAP-based User Activity Monitoring System** designed to track user login/logout activities, authentication failures, and send alert emails for suspicious or abnormal behaviors in the SAP system.

This package provides backend programs, database tables, and service-ready objects for monitoring and auditing user activities.

---

## 🚀 Tech Stack

- SAP ABAP
- SAP GUI
- Ehancement
- Bacthjob
- ABAP Dictionary
- ABAP Reports / Programs
- CDS Views
- Service Definitions / Bindings
- Access Controls
- Behavior Definitions
- Metadata Extensions
- Message Class
- Class / Methods

---

## 🧩 Main Features

### 👤 User Activity Logging
- Track user login time / logout time
- Record activity duration
- Store activity logs in database tables

### 🔐 Authentication Monitoring
- Record failed login attempts
- Detect suspicious login behaviors
- Store authentication logs for auditing

### 📊 Service-Ready Architecture
- CDS Views for data exposure
- Service Definitions for APIs
- Service Bindings for OData/Fiori integration
